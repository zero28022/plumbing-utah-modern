# Plumbing Utah - Modern Website Rebuild

A fully responsive, modern, and engaging website for Plumbing Utah built with HTML5, CSS3, and vanilla JavaScript.

## 🎨 Design Features

### Modern Elements
- **Glassmorphism** - Beautiful frosted glass effect cards
- **Smooth Animations** - Elegant transitions and scroll effects
- **Responsive Design** - Optimized for all screen sizes
- **Minimalist Layout** - Clean, uncluttered interface
- **Gradient Accents** - Modern color schemes with smooth gradients

### Key Sections
- **Hero** - Eye-catching landing section with CTAs
- **Services** - 6 service cards with hover effects
- **Why Us** - 6 reason cards with glassmorphism effect
- **About** - Company information with feature highlights
- **Testimonials** - Customer reviews section
- **Contact** - Contact form and contact information
- **Footer** - Comprehensive footer with links

## 🚀 Quick Start

1. **Clone or Download** this project
2. **Open `index.html`** in a web browser
3. **Customize** with your own content, images, and branding
